Domain names and ICANN

Domain names are the sequence of characters used by humans that delineate a website's IP address. Rather than reading a string of numbers, it is more user friendly to simply type in ‘google.com’.

IP Addresses, Packets and Routing

Every wonder how information is sent over the internet without being lost? This comes down to fault tolerance and redundancy, aspects of routing that allow data to be sent in ways that prevent information from being lost.

DNS

The DNS, or Domain Name System helps to translate IP addresses (the format of numbers that computers use) into domain names that are readable to humans.

HTTP and HTTPS protocols

HTTP is the language that is used between web browsers and servers, which is made mostly of ‘get’ requests that ask for information. HTTPS makes sure that HTTP requests are scure.

W3C and HTML and CSS

The W3C or World Wide Web Consortium is the main organization that creates the global standards for the world-wide web. HTML, or Hyper Text Markup language, Is the standard language used for creating web pages. CSS designates how HTML elements are displayed.
