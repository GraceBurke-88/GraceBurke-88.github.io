W3C, HTML, and CSS

W3C

The main organization that creates the global standards for the world-wide web
It was founded in 1994 and the current leader is Tim Berners-Lee, the members include full-time staff, industry experts, and other organizations.
The W3C also is involved in education and takes part in service, they also develop software and hold open discussion regarding the web.
The W3C works to keep protocols current, for example when browsers utilize W3C standard, the websites will appear the same over different web browsers.
Standards developed by the Consortium include: CGI CSS DOM HTML HTTP XHTML XML

HTML

Hyper Text Markup language
The standard language used for creating web pages
HTML describes the structure of the webpagee, which is made up of various elements

CSS

Cascading Style Sheets
CSS is the language utilized to style an html document
Thus it describes how the various html elements should show up and how the elements should be displayed.
Can save time since it can describe several web pages all at once.
